An Archetype represents a stable creative identity pattern, not a personality type or behavioral score.

## Archetype Definition
An Archetype describes how creative drive, expression, and meaning-making converge over time.

## Required Fields
- archetypeId
- name
- sphere
- family
- coreDrive
- creativeExpression
- distortionRisk
- integrationNote

## Constraints
- Archetypes are interpretive, not evaluative.
- No archetype is ranked above another.
- Language must describe tendencies, not fixed traits.
